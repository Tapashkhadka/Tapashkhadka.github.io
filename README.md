# Tapash Khadka — Personal Website

A fast, polished static portfolio site for showcasing résumé, work, projects, skills, and contact links.

## Built with

- HTML
- CSS
- Vanilla JavaScript
- No external dependencies

## Local preview

```bash
python3 -m http.server 4173
```

Then open: <http://localhost:4173>

## Customize before publishing

- Replace placeholder email in `index.html`
- Replace LinkedIn URL in `index.html`
- Add a real `resume.pdf` at the repo root
- Update project cards with real screenshots, links, and outcomes
- Replace the placeholder testimonial with a real quote

## Local-model note

Initial design direction was generated locally with Ollama using `qwen3:14b`; implementation is intentionally static and lightweight.
